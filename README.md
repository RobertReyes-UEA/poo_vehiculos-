# Proyecto POO en Python – Vehículos

## Descripción
Este proyecto es un programa desarrollado en **Python** que aplica los principios de la **Programación Orientada a Objetos (POO)**.  
Cumple con los requisitos académicos de implementar **herencia, encapsulación y polimorfismo**, además de una correcta organización del proyecto por carpetas.

El programa simula la gestión básica de un vehículo, específicamente un automóvil.

---

## Conceptos de POO Aplicados
- **Herencia:**  
  La clase `Auto` hereda de la clase base `Vehiculo`.

- **Encapsulación:**  
  El atributo `__velocidad` está protegido y solo se accede mediante métodos públicos.

- **Polimorfismo:**  
  El método `describir()` es sobrescrito en la clase `Auto`.
